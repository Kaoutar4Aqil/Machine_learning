# Machine_learning
This project focuses on building a machine learning model to classify emails as spam or ham (non-spam) using Natural Language Processing (NLP) techniques.
The dataset contains emails labeled as:
ham (legitimate emails)
spam (unwanted or malicious emails)
The dataset is imbalanced:
~85% ham
~15% spam
The project demonstrates an end-to-end machine learning pipeline for spam detection, including:
Text preprocessing
Feature extraction
Handling imbalanced data
Model training and evaluation
Visualization for interpretability
